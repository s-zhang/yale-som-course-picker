MySOMClasses - Structured Data Examples

Note: These JSON-LD structured data snippets are automatically embedded in the page HTML to help search engines understand and index course information following Google's Course structured data guidelines.

1. WebSite Structured Data

Defines the website and enables search functionality in Google results:

{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "MySOMClasses",
  "url": "https://mysomclasses.com",
  "description": "Browse, filter, and schedule courses at Yale School of Management",
  "potentialAction": {
    "@type": "SearchAction",
    "target": {
      "@type": "EntryPoint",
      "urlTemplate": "https://mysomclasses.com/?search={search_term_string}"
    },
    "query-input": "required name=search_term_string"
  }
}

Testing the Structured Data

You can test the structured data using:

  • Google Rich Results Test
  • Schema.org Validator
  • Google Search Console (after deployment)

Benefits

  • Rich Search Results: Courses may appear with enhanced information in Google search
  • Better Discovery: Improved indexing helps students find relevant courses
  • Knowledge Graph: Information may appear in Google's Knowledge Graph
  • Search Features: Enable special search features like course filters and site search
